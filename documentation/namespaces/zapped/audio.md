# audio

## Example:

```lua
audio.play_voice("C:\\zapped\\sounds\\song.wav")
```

## Functions:

### play_sound

Plays a sound locally.

| Argument | Type   | Required |
| -------- | ------ | -------- |
| filename | string | +        |

### play_voice

Plays a WAV file through your in game voice chat.

| Argument     | Type   | Required |
| ------------ | ------ | -------- |
| wav filename | string | +        |

### stop_playback

Stops all audio playback.
